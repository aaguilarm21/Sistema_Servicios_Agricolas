ENTIDAD-RELACIÓN DEL PROYECTO: SISTEMA DE SERVICIOS AGRÍCOLAS

1. ENTIDAD: Usuario (User)
   - ID (Clave Primaria)
   - nombre_usuario (username)
   - contraseña (password)
   - correo_electrónico (email)
   - nombre (first_name)
   - apellido (last_name)
   - es_personal (is_staff)
   - está_activo (is_active)
   - fecha_de_registro (date_joined)

2. ENTIDAD: Perfil de Usuario (UserProfile)
   - id (CP)
   - usuario (Usuario, Uno a Uno)
   - código (Texto, único)
   - puesto (Texto)
   - creado_el (Fecha y Hora)
   - actualizado_el (Fecha y Hora)

3. ENTIDAD: Proveedor
   - id (CP)
   - código (Texto, único)
   - nit (Texto)
   - razón_social (Texto)
   - nombre_propietario (Texto)
   - régimen_tributario (Texto)
   - tipo_factura (Texto)
   - días_crédito (Entero)
   - creado_el (Fecha y Hora)
   - actualizado_el (Fecha y Hora)

4. ENTIDAD: Empleado
   - id (CP)
   - empresa (Texto)
   - empleado (Texto)
   - no_cui (Texto)
   - puesto (Texto)
   - nombre_puesto (Texto)
   - creado_el (Fecha y Hora)
   - actualizado_el (Fecha y Hora)

5. ENTIDAD: Maquinaria
   - id (CP)
   - código_máquina (Texto, único)
   - combustible (Texto)
   - id_proveedor (Texto - Relación lógica con Proveedor.código)
   - tipo_máquina (Texto)
   - marca_máquina (Texto)
   - serie_máquina (Texto)
   - placa_matrícula (Texto)
   - observaciones (Texto Largo)
   - creado_el (Fecha y Hora)
   - actualizado_el (Fecha y Hora)

6. ENTIDAD: Bodega
   - id (CP)
   - código (Texto, único)
   - nombre_bodega (Texto)
   - unidad_medida (Texto)
   - capacidad (Texto)
   - observaciones (Texto Largo)
   - creado_el (Fecha y Hora)
   - actualizado_el (Fecha y Hora)

7. ENTIDAD: Artículo (Articulo)
   - id (CP)
   - código_artículo (Texto, único)
   - descripción (Texto)
   - unidad_medida (Texto)
   - categoría (Texto)
   - stock (Decimal)
   - creado_el (Fecha y Hora)
   - actualizado_el (Fecha y Hora)

RELACIONES:
- Perfil de Usuario (1) <---> (1) Usuario : Relación uno a uno para extender los datos del usuario.
- Maquinaria (N) ----> (1) Proveedor : Relación lógica a través del campo 'id_proveedor' (vinculado al código del proveedor).
